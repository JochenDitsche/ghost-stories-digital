# 👻 Ghost Stories - Digital

Ein browser-basiertes Brettspiel inspiriert vom klassischen Ghost Stories.

## 🎮 Spielen

**Online:** https://jochenditsche.github.io/ghost-stories-digital

## 📁 Projektstruktur

```
ghost-stories-digital/
├── index.html          # Hauptseite
├── css/
│   └── style.css       # Styles und Layout
├── js/
│   └── game.js         # Spielmechanik und Logik
└── README.md           # Diese Datei
```

## 🎯 Features

- **4 einzigartige Mönche** mit speziellen Fähigkeiten
- **Yin-Yang Phasensystem**: Geister ziehen, dann Mönche agieren
- **Buddha-Platzierung** zum Schutz des Dorfes
- **Tao-Token System** mit Beute-Mechanik
- **Wu-Feng Bosskampf** wenn das Deck leer ist
- **Responsive Design** für Desktop und Mobile

## 🛠️ Technik

Reines HTML5 + CSS3 + Vanilla JavaScript. Kein Build-Prozess nötig.
